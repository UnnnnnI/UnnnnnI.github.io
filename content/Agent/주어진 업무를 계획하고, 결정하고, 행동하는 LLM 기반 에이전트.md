
에이전트: 지능 객체로 일정 수준의 자율성을 가진 인공지능 시스템
		환경을 감지하고, 결정을 내리며 행동을 취하는 시스템

LLM이 등장하며 언어 이해와 추론 능력을 보여주면서, 에이전트의 능력이 강화됨

LLM 기반의 에이전트 구성 요소
1) 계획: 당연히 일을 계획하고 하면 능률이 좋다,,!
	   작은 목표로 분할 / 사고의 연쇄 / 자기 성찰과 비판 / 과거 행동에 대한 성찰 등...
2) 기억: 단기 기억과 장기 기억
	   외부 벡터 스토리지와 빠른 검색을 활용
3) 도구: 에이전트가 호출할 수 있는 다양한 도구들
		LLM은 데이터 기반으로 답을 내놓으니까,, 행동을 할 도구와 연결이 필요함
		ex: 데이터 분석 에이전트 → 통계 분석 SW, 혹은 머신러닝 모델 등을 호출하여 데이터를 처리
4) 실행: 구체적인 행동을 실행
	   도구를 호출하여 작업을 완료함

에이전트는 하나만 존재하는 것이 아님
다중의 에이전트가 협업을 통해 함께 작업을 수행하는 방향으로 발전함
 - 상위 계층 에이전트는 젼체 조정 작업 수행 (의사 결정, 목표 설정 등)
 - 하위 계층 에이전트는 구체적인 작업 수행 (데이터 수집이나 세부 작업 처리 등)
